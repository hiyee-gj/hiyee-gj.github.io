모듈

  • 필요한 것을 모아둔 파일
  • 확장자 .py

모듈 생성

파일명 : theater_module.py

# 일반 가격
def price(people):
    print(f"{people}명 가격은 {people * 10000}원 입니다.")

# 조조할인 가격
def price_morning(people):
    print(f"{people}명 조조할인 가격은 {people * 6000}원 입니다.")

# 군인할인 가격
def price_soldier(people):
    print(f"{people}명 군인할인 가격은 {people * 4000}원 입니다.")

모듈 활용

  • 예시1
import theater_module
theater_module.price(3) # 3명이서 영화 보러 갔을 때 가격
theater_module.price_morning(4) # 4명이서 조조할인 영화 보러 갔을 때 가격
theater_module.price_soldier(5) # 5명의 군인이 영화 보러 갔을 때 가격

출력:
3명 가격은 30000원 입니다.
4명 조조할인 가격은 24000원 입니다.
5명 군인할인 가격은 20000원 입니다.
  • 예시2(모듈명에 별명을 붙여 사용하는 법)
import theater_module as mv
mv.price(3)
mv.price_morning(4)
mv.price_soldier(5)

출력:
3명 가격은 30000원 입니다.
4명 조조할인 가격은 24000원 입니다.
5명 군인할인 가격은 20000원 입니다.
  • 예시3(from으로 호출해서 사용하기)
from theater_module import *
price(3)
price_morning(4)
price_soldier(5)

출력:
3명 가격은 30000원 입니다.
4명 조조할인 가격은 24000원 입니다.
5명 군인할인 가격은 20000원 입니다.
  • 예시4(필요한 함수만 불러와서 사용하기)
from theater_module import price, price_morning
price(5)
price_morning(6)

출력:
5명 가격은 50000원 입니다.
6명 조조할인 가격은 36000원 입니다.
  • 예시5(불러온 함수에 별명을 붙여 사용하기)
from theater_module import price_soldier as ps
ps(5)

출력:
5명 군인할인 가격은 20000원 입니다.

패키지

  • 모듈들을 모아둔 집합
1. 패키지 생성 newfolder로 원하는 패키지 생성(예시: travel)
2. 패키지 안에 newfile로 모듈 생성(예시: thailand.py, vietnam.py, __init__.py)

thailand.py 내용

class THailandPackage:
    def detail(self):
        print("[태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원")

vietnam.py 내용

class VietnamPackage:
    def detail(self):
        print("[베트남 패키지 3박 5일] 다낭 효도 여행 60만원")

practice.py에서 import로 사용

import travel.thailand
trip_to = travel.thailand.THailandPackage()
trip_to.detail()

출력: [태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원
  • 주의할 점 import travel. 뒤의 내용은 모듈이나 패키지만 올 수 있음.(클래스나 함수는 바로 올 수 없음)
  • from으로 가져올 경우에는 사용할 수 있음
from travel.thailand import THailandPackage
trip_to = THailandPackage()
trip_to.detail()

출력: [태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원

from travel import vietnam
trip_to = vietnam.VietnamPackage()
trip_to.detail()

출력: [베트남 패키지 3박 5일] 다낭 효도 여행 60만원

__all__

# random 모듈을 사용할 때
from random import * 과 같이 생성한 모듈에 적용하면

from travel import *

출력: NameError: name 'vietnam' is not defined

오류가 발생한다
  • 이유는 생성한 모듈의 공개범위(공개할 내용과 비공개할 내용)을 설정하지 않았기 때문이다

공개범위 설정

생성해 뒀던 __init__.py에서
__all__ = ["vietnam","thailand"] 설정

from travel import *
trip_to = vietnam.VietnamPackage()
trip_to.detail()
trip_to = thailand.THailandPackage()
trip_to.detail()

출력:
[베트남 패키지 3박 5일] 다낭 효도 여행 60만원
[태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원

모듈 직접 실행

  • 모듈이 잘 동작하는지 확인하기 위함
  • thailand.py 에서 직접 실행
class THailandPackage:
    def detail(self):
        print("[태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원")

if __name__ == "__main__": # name이 main이면 thailand 모듈을 직접 실행
    print("Thailand 모듈을 직접 실행")
    print("이 문장은 모듈을 직접 실행할 때만 실행돼요")
    trip_to = THailandPackage()
    trip_to.detail()
else:
    print("Thailand 외부에서 모듈 호출")

출력:
Thailand 모듈을 직접 실행
이 문장은 모듈을 직접 실행할 때만 실행돼요
[태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원
  • practice.py 에서 외부 실행
from travel import *
trip_to = thailand.THailandPackage()
trip_to.detail()

출력:
Thailand 외부에서 모듈 호출
[태국 패키지 3박 5일] 방콕, 파타야 여행 (야시장 투어) 50만원

  • 패키지, 모듈의 위치 파악

import inspect

/Library/Frameworks/Python.framework/Versions/3.9/lib/python3.9/random.py

pip install

  • pip로 패키지 install하기
  • Google에서 pypi 검색하면 수많은 패키지가 존재한다. 여기 없는 경우 만들어 쓰도록 하자
  • 예시에서는 웹스크래핑에서 유명한 beautifulsoup을 설치해 본다
vscode terminal에 pypi에서 긁어온 pip install beautifulsoup4을 입력했더니 not found command라는 오류가 발생한다

검색해 본 결과 pip대신 pip3로 명령내려주면 실행됨

pip3 install beautifulsoup4

  • Google에서 list of python buitins을 검색하면 Built-in Functions에서 사용가능한 내장 함수들이 나온다

  • ex1) input : 사용자 입력을 받는 함수

  • ex2) dir : 어떤 객체를 넘겨줬을 때 그 객체가 어떤 변수와 함수를 가지고 있는지 표시

import random # 외장함수
print(dir())

출력: ['__annotations__', '__builtins__', '__cached__', '__doc__', '__file__', '__loader__', '__name__', '__package__', '__spec__', 'random']

import random
print(dir(random))

['BPF', 'LOG4', 'NV_MAGICCONST', 'RECIP_BPF', 'Random', 'SG_MAGICCONST', 'SystemRandom', 'TWOPI', '_Sequence', '_Set', '__all__', '__builtins__', '__cached__', '__doc__', '__file__', '__loader__', '__name__', '__package__', '__spec__', '_accumulate', '_acos', '_bisect', '_ceil', '_cos', '_e', '_exp', '_floor', '_inst', '_log', '_os', '_pi', '_random', '_repeat', '_sha512', '_sin', '_sqrt', '_test', '_test_generator', '_urandom', '_warn', 'betavariate', 'choice', 'choices', 'expovariate', 'gammavariate', 'gauss', 'getrandbits', 'getstate', 'lognormvariate', 'normalvariate', 'paretovariate', 'randbytes', 'randint', 'random', 'randrange', 'sample', 'seed', 'setstate', 'shuffle', 'triangular', 'uniform', 'vonmisesvariate', 'weibullvariate']

외장함수

  • Google에서 list of python modules을 검색하면 Python Module Index에서 사용가능한 외장 함수들이 나온다
  • ex1) glob : 경로 내의 폴더 / 파일 목록 조회 (윈도우 dir)
import glob
print(glob.glob("*.py")) # 확장자가 py 인 모든 파일

출력:
['practice.py', 'helloworld.py', 'theater_module.py']
  • ex2) os : 운영체제에서 제공하는 기본 기능
import os
print(os.getcwd()) # 현재 디렉토리 표시
# 등등 ...

출력: /Users/hiyee/Desktop/PythonWorkspace
  • ex3) time : 시간 관련 함수
import time
print(time.localtime())
print(time.strftime("%Y-%m-%d %H:%M:%S"))

출력: 2021-03-08 16:33:52
  • ex4) datetime
import datetime
print("오늘 날짜는 ", datetime.date.today())

# timedelta : 두 날짜 사이의 간격
today = datetime.date.today() # 오늘 날짜 저장
td = datetime.timedelta(days=100) # 100일 저장
print("우리가 만난지 100일은", today + td) # 오늘부터 100일 후

출력:
오늘 날짜는  2021-03-08
우리가 만난지 100일은 2021-06-16
